# MyBB Security Checklist

- Enable MyBB Two-Factor Authentication for every administrator account.
- Use a password manager with unique passwords for forum, email, GitHub, and Render.
- Use private browsing or a separate browser session for ACP access.
- Keep trusted admins to the minimum and review ACP permissions regularly.
- Review plugin/theme code before installation and keep extensions to a minimum.
- Verify backups periodically by restoring them into a test environment.
